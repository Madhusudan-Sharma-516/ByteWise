First Java Program: Hello, World!

Java me, हर program ek class ke अंदर होता है। हमारा पहला program, जिसे "Hello, World!" कहा जाता है, बहुत simple है aur ये तुम्हें Java ki basic structure समझने me help karega। Yeh program sirf screen par "Hello, World!" print करता है।

Example :

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
Every Token Explained
नीचे दी गई तालिका में, प्रोग्राम में उपयोग किए गए हर शब्द और प्रतीक (symbol) की व्याख्या की गई है।
Token Explanation (Hindlish)
public Ye ek keyword (access modifier) hai jo class या method को कहीं से भी access करने लायक बनाता है।
class Ye ek keyword hai जिसका use class declare करने के लिए होता है। Ye objects बनाने का ek blueprint है।
HelloWorld Ye class का नाम है, और iska naam filename (HelloWorld.java) के साथ match होना चाहिए।
{ } Ye braces ek code block की शुरुआत और अंत दिखाते हैं।
static Ye ek keyword hai जो main method को बिना object बनाए run करने देता है।
void Ye method का return type hai, इसका मतलब method कोई value return नहीं करेगा।
main Ye main method का नाम hai, और JVM इसे program का starting point मानता है।
() Ye parentheses method में parameters pass करने के लिए use होते हैं।
String[] args Ye ek array of strings hai जो command-line arguments को accept करता है।
System Ye Java की ek predefined class hai जो system resources तक access देती है।
. Ye dot operator class के members को access करने के लिए use होता है।
out Ye System class का ek static member hai जो standard output (console) को represent करता है।
println Ye ek method hai जो console पर text print करता है और ek नई line जोड़ता है।
; Ye semicolon ek statement के end को दिखाता है।
"Hello, World!" Ye ek string literal hai, मतलब वही text जो console पर print होगा।
